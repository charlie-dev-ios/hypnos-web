- **UI Library**: React 19.2.1
- **言語**: TypeScript 5.9.3 (strict mode)
- **スタイリング**: Tailwind CSS 4.x
- **UIコンポーネント**: shadcn/ui (Radix UIベース)

- **アイコン**: lucide-react
